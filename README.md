# json-parse-verification

### Usage
- Confirmation by General decode
```
$ go run basic.go
```

- Confirmation by json.NewDecoder
```
$ go run decode.go
```

- Confirmation by json.Unmarshal
```
$ go run unmarshal.go
```

- Confirmation by interface
```
$ go run interface.go
```

- Confirmation by jason
```
$ go run jason.go
```
